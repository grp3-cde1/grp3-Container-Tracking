# Tutorial

Ein Teil des Projekts ist ein Tutorial für Python-Einsteiger. Das Tutorial orientiert sich an den eigenen Anwendungen und erklärt Python nicht abstrakt, sondern anhand realer Aufgaben aus dem Projekt.

## Schritt 1: Einfache Single-File-Version
Zunächst wird eine Retrospektive-Applikation in einem einzigen Python-File umgesetzt. Dadurch lassen sich die Grundlagen von Python verständlich einführen.

Behandelte Themen:
- Variablen
- Datentypen
- Listen
- Dictionaries
- Schleifen
- Bedingungen
- Funktionen
- Datei-Verarbeitung mit CSV

## Schritt 2: Strukturierung und OOP
Anschliessend wird die erste Version überarbeitet und in eine besser strukturierte Form gebracht. Dabei werden Konzepte der objektorientierten Programmierung eingeführt.

Behandelte Themen:
- Klassen
- Objekte
- Attribute
- Methoden
- Modularisierung
